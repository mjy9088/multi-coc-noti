# Overview

`packages/ui/src/styles`

## Purpose

Shared visual foundation for `@multi-coc/ui`. Feature styles consume these semantic variables and component classes instead
of introducing raw colors, radii, shadows, z-index values, or duplicate interaction geometry.

## Key files

- `tokens.css` — semantic colors, spacing, radius, shadow, and layer variables.
- `foundations.css` — browser reset and global element foundations.
- `components.css` — primitive and composition presentation.
- `index.css` — public stylesheet entry point.
