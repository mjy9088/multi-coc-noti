# Overview

`packages/ui/src/components/data-display`

## Purpose

Reusable content presentation primitives for product records and request states. These components define structure and
accessibility but do not know village, upgrade, notification, or route semantics.

## Areas

- cards and data-list rows;
- statistics and statistic grids;
- entity identity headers;
- empty, loading, error, and stale-data states.
