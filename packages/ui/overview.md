# Overview

`packages/ui`

## Purpose

Owned generic React design-system package. It provides semantic tokens, accessible primitives, layout and scroll behavior,
feedback, overlays, forms, and concise composition APIs without game-specific terminology.

## Subfolders

- `src/` — public exports, component implementations, utilities, and styles.

Dashboard and UI Lab consume this package; it must not import product routes, translations, queries, or domain models.
