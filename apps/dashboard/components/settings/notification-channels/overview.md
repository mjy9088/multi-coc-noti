# Overview

`apps/dashboard/components/settings/notification-channels`

## Purpose

Notification-channel settings feature. The tab composes the saved-channel collection and registration form without owning
request orchestration.

## Key files

- `notification-channels-tab.tsx` — channel collection and registration composition.
- `notification-channel-item.tsx` — one saved channel and its delete action.
- `notification-channel-form.tsx` — Bark channel registration fields.
