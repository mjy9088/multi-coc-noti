# Overview

`apps/dashboard/components/settings/upgrade-alerts`

## Purpose

Upgrade-alert settings feature. The tab owns collection selection and draft routing; each item owns one upgrade's
presentation, policy controls, validation, and actions.

## Key files

- `upgrade-alerts-tab.tsx` — active-upgrade collection and draft coordination.
- `upgrade-alert-item.tsx` — one upgrade's alert policy editor and actions.
