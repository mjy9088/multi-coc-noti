# Overview

`apps/ui-lab/app/compositions`

## Purpose

Disposable composition workbench for comparing screen hierarchy and component placement before production integration.

## Subfolders

- `dashboard/` — Dashboard density and ordering studies.
- `history/` — History filtering and result studies.
- `import/` — paste and review studies.
- `settings/` — Settings navigation and master/detail studies.

`composition-workbench.tsx` provides the shared review frame.
