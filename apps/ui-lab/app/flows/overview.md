# Overview

`apps/ui-lab/app/flows`

## Purpose

Fixture-driven product-flow simulations for reviewing selection, loading, error, empty, large-data, and responsive states.

## Subfolders

- `dashboard/` — many-village Dashboard scenarios.
- `history/` — history list and filter scenarios.
- `import/` — paste, review, and result scenarios.
- `settings/` — settings navigation and village-editor scenarios.

`flow-workbench.tsx` owns shared scenario and viewport controls.
